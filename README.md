# 📊 Financial Performance Dashboard

## Overview

This interactive Financial Performance Dashboard was developed using **Power BI** to monitor financial performance through key business indicators and interactive visualizations.

The dashboard transforms financial data into meaningful insights, supporting business planning and strategic decision-making.

---

## Business Objectives

* Monitor financial performance.
* Compare actual results against budget.
* Analyze profitability by region.
* Track revenue trends over time.
* Support executive decision-making.

---

## Key Performance Indicators (KPIs)

* Total Revenue
* Total COGS
* Total Profit
* Profit Margin
* Budget Achievement

---

## Dashboard Features

* Revenue Trend
* Revenue by Region
* Revenue by Sales Channel
* Revenue by Product Category
* Profit Analysis
* Budget vs Actual
* Executive Key Insights
